###Setup:
Make sure you have npm ([LINK]), bower ([LINK]) and Compass ([LINK]) installed

#####Install NPM components from /package.json
```
$ npm install
```
#####Install Ruby Gems from /Gemfile
```
$ bundle install --path=gems
```

###Start local development server:
```
$ grunt serve
```

###Build application:
```
$ grunt build
```
